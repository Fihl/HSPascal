HSPascal

HSPascal by Christen Fihl

The history of HSPascal started back in 1990. Supported computers were the Atari and Amiga computers

New century, new compiler, new platform

HSPascal will now be supporting the Palm(r) handheld computer

  • Written 100% in Borland Delphi (Pascal) for Windows
  • Get a program like DoodleHS.pas translated into DoodleHS.prc in 1.0 second
  • Resources imported or right inline in Pascal source
  • Floating point support without MathLib and the hassle
  • Strings the easy way
  • See more features and examples

Commercials

HSPascal is a commercial product. Or will be, as it is not yet finished.
The estimated price will be about $75 from PalmGear.

An official release date does not exist yet, but hopefully not far away.
The first version will be made for sale in a limited number. I expect beta testers and myself to validate the code generation. Still I do not want to flood faulty packages all over.
Test versions has since Nov-2000 be available to a selected few for test, review and hands-on purpose.
Test programs compiled with this test version can be found on PalmGear (look for HSPascal).

I have been waiting for Palm Inc to allow me redistribution-rights for my Pascal version of the header files (the *.H from Palm sdk).
I just got the papers !
